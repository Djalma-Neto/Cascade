===================================== REQUISITOS DE INSTALAÇÃO =====================================

Para executar os seguntes scripts deve ter instalado em sua máquina o python

> Windows: https://www.python.org/downloads/windows/

> Linux: Geralmente já vem instalado basta atualizar: 
Para isso abra o Terminal e digite os seguintes códigos:

> sudo add-apt-repository ppa:deadsnakes/ppa
> sudo apt update
> sudo apt install python3.10

posteriormente, no Window, abra o CMD no mesmo diretório onde se encontra o arquivo "get-pip.py", deve-se executa-lo  para instalar o pip que é o gerenciador de pacotes do python.
OBS.: A depender da versão o comando "python" pode ser alterado por: "py" ou "python3"

EX1.: > python get-pip.py
EX2.: > py get-pip.py
EX3.: > python3 get-pip.py

depois de instalado deve-se abrir o terminal ou CMD no diretório onde se encontra o arquivo a ser executado
EX.: ".\Trabalho_01"

executar o comando de instalação para que, dessa forma, as bibliotecas utilizadas sejam
instaladas

EX.: > python -m pip install -r requirements.txt

posteriormente executar o arquivo desejado.
EX.: > python nomeArquivoDesejado.py
